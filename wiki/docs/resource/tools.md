## 在线工具

- LaTeX 表格：
  - [:link: Tables Generator](https://www.tablesgenerator.com/)
- LaTeX 数学公式编辑：
  - [:link: LaTeX 公式编辑器](https://www.latexlive.com/)
  - [:link: Mathcha.io](https://www.mathcha.io/)（可视化编辑和函数图像，在线版本免费，离线版本收费）
- 符号查询识别：
  - LaTex 数学符号查询（仅限英文） [:link: latex.guide](https://latex.guide/)
  - 手写字符（特殊符号）识别 [:link: detexify.kirelabs.org](http://detexify.kirelabs.org/classify.html)
  - 速查表 [:link: LaTeX cheatsheet](https://quickref.me/latex), [:link: 中文页面](https://quickref.me/zh-CN/docs/latex.html)

## 网页公式渲染

- [:link: MathJax](https://www.mathjax.org/): 浏览器网页显示数学公式的 JavaScript 渲染引擎，支持语言 LaTex、MathML、AsciiMath。
  - 来自 StackExchange 中教程 [MathJax basic tutorial and quick reference](https://math.meta.stackexchange.com/questions/5020/mathjax-basic-tutorial-and-quick-reference)
- [:link: KaTex](https://katex.org/): 数学公式渲染器 ，相比 MathJax 更轻量级。
  - [:link: 文档 - Supported Functions](https://katex.org/docs/supported.html)
